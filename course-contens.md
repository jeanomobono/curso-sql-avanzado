# Advanced SQL Course

This course will be about advanced topics about SQL language on Oracle Databases. I will use Oracle 23ai Free as database with the VUELOS dataset and use VS Code as the presentantion tool, which already has the Oracle SQL Developer extension, I would like to use SQL Notebooks as a way to present both information about the SQL statements and demos.

## Course content

I found this [list](https://softwaredominos.wpcomstaging.com/home/software-engineering-and-computer-science/oracle-sql-tutorial-your-guide-to-advanced-query-techniques/) on internet and I think it covers most of what the students need. Use it as a base for the content of our course.
